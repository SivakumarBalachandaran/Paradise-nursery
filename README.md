# Paradise Nursery Shopping Application

## Project Name: Paradise Nursery

A React-based e-commerce shopping cart application for an online plant shop.

## Features

- Browse houseplants grouped by category
- Add plants to a shopping cart
- View and manage cart items (increase/decrease quantity, remove items)
- Dynamic cart count in the navigation bar
- Responsive design with a nature-themed UI

## Tech Stack

- React (with Vite)
- Redux Toolkit (for cart state management)
- CSS3

## Getting Started

```bash
npm install
npm run dev
```
